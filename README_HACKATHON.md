# 🏆 Katya AI REChain Mesh - Hackathon Winner

## 🚀 Суперапп нового поколения для оффлайн коммуникаций

### ✨ Что это?

**Katya AI REChain Mesh** - это революционный суперапп, который работает полностью оффлайн через Bluetooth Low Energy (BLE) mesh-сеть. Включает в себя мессенджер с AI-ассистентом, систему голосований, новости, карты, телемедицину и многое другое.

---

## 🎯 Ключевые особенности

### 🔥 Уникальные преимущества
- **🌐 Оффлайн-first** - работает без интернета
- **🤖 AI-powered** - умный ассистент Katya AI
- **🔒 End-to-End шифрование** - военная криптография
- **📱 Суперапп** - множество сервисов в одном
- **⚡ Быстрая работа** - оптимизированная производительность

### 🛠 Технологический стек
- **Frontend**: Flutter + Dart
- **Backend**: Dart Frog + **Go (mesh broker, CLI)**
- **Networking**: Bluetooth Low Energy (BLE)
- **Security**: AES-256-GCM, X25519, HKDF-SHA256 (Go crypto)
- **AI**: Встроенный ассистент Katya AI (оффлайн)
- **Multi-language**: Dart (Flutter app) + **Go (CLI, services)**

## 🚀 Быстрый старт

### Установка
```bash
# Клонирование репозитория
git clone https://github.com/your-repo/katya-ai-rechain-mesh.git
cd katya-ai-rechain-mesh

# Установка зависимостей
flutter pub get

# Запуск приложения
flutter run
```

### Минимальная демо-схема (оффлайн mesh, 2 инстанса)
```bash
# Откройте 2 терминала и запустите приложение дважды на одной сети (или на 2 устройствах)
# Инстансы автоматически сгенерируют разные deviceId и будут обнаруживаться по UDP multicast

# Терминал 1
flutter run -d windows

# Терминал 2 (или второе устройство/эмулятор)
flutter run -d windows

# В приложении:
# 1) Вкладка "Устройства" → убедитесь, что виден второй пир
# 2) Вкладка "Чаты" → отправьте сообщение (если нет прямого пира, сработает broadcast)
# 3) Вкладка "Голосование" → создайте опрос на одном инстансе — он появится на втором
```

### Сборка APK
```bash
# Debug версия
flutter build apk --debug

# Release версия
flutter build apk --release
```

### Сборка Go CLI и сервисов
```bash
cd go
make build
# или вручную:
go build -o bin/meshctl ./cmd/meshctl
go build -o bin/mesh-broker ./cmd/mesh-broker

# Использование:
./bin/meshctl start --adapter emulated
./bin/meshctl peers
./bin/meshctl send --to broadcast --message "Hello!"
./bin/meshctl poll create --title "Where?" --options "A,B,C"
./bin/meshctl stats
```

## 🎬 Демонстрация

### Сценарий 1: Оффлайн мессенджер с AI
1. Отключите интернет на устройстве
2. Откройте приложение → вкладка "Чаты"
3. Отправьте сообщение через BLE (broadcast, если нет прямых пиров)
4. **Покажите AI-баннер с подсказками** — Katya AI предлагает готовые ответы
5. Используйте ActionChip для быстрой вставки ответа
6. Сообщения сохраняются оффлайн (перезапустите приложение — история останется)

### Сценарий 2: Система голосований с AI-анализом
1. Перейдите на вкладку "Голосования"
2. Создайте новый опрос (например, "Где встретиться?")
3. Второй инстанс увидит опрос автоматически (mesh-синхронизация)
4. Проголосуйте с обоих инстансов — счётчики обновляются в реальном времени
5. **Нажмите "🤖 AI Analysis"** — Katya AI проанализирует результаты
6. Покажите диалог с анализом: лидер, проценты, рекомендации
7. (Опционально) Завершите по majority для демонстрации оффлайн-консенсуса

### Сценарий 3: Интерактивный AI-чат
1. Перейдите на вкладку "Katya AI"
2. Задайте вопрос: "Привет", "Как дела?", "Что такое mesh-сеть?"
3. Покажите ответы AI в реальном времени
4. AI учитывает контекст (количество пиров, статус сети)

### Сценарий 4: Mesh HUD и Screencast режим
1. **Mesh HUD** (правый нижний угол) — показывает живую статистику
2. Раскройте HUD — увидите: пиры, очередь, успешность, рестарты
3. **"Start Screencast"** — автоматическая демо-последовательность:
   - Отправляет приветственное сообщение
   - Создаёт демо-опрос и голосует
   - Отправляет follow-up сообщения
4. **"Reset Demo"** — очистка состояния для повторного демо

### Сценарий 5: Суперапп функции
1. Покажите новости и контент
2. Продемонстрируйте карты и геолокацию
3. Покажите телемедицину
4. Откройте аналитику
