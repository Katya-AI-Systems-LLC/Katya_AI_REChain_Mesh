# macOS Platform Development

This directory contains the macOS-specific implementation for the Katya AI REChain Mesh project using Flutter.

## Structure

- **Flutter/**: Flutter-specific configurations and code for macOS.
- **Runner/**: macOS runner and project files for Xcode.
- **build.gradle**: Gradle build script.

## Development Setup

1. **Prerequisites**:
   - Install Flutter SDK.
   - Install Xcode (for macOS development).
   - Ensure you are on a macOS machine.

2. **Building the App**:
   - Run `flutter pub get` to install dependencies.
   - Use `flutter build macos` to build for macOS.
   - Open in Xcode: `open macos/Runner.xcworkspace`.
   - Build and run.

3. **Features**:
   - Blockchain integration via Flutter plugins.
   - Gaming and IoT modules adapted for desktop.
   - Social features in Dart.

## Customization

- Update `lib/` for shared logic.
- Modify macOS-specific code in `Runner/` for native features like menu bars or system integrations.

## Documentation

See Flutter's macOS documentation and the project's main README.
