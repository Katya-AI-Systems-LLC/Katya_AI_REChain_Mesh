# Mesh Quantum App (RU)

Оффлайн mesh-мессенджер, квантовая голосовалка, AI агент — всё без интернета. 

* BLE, Wi-Fi, Nearby, голосование, AI, CLI, Blockchain
* Любая платформа: Android, iOS, Win, Mac, Linux, Web, Aurora
* Скринкаст: https://your-screencast-link

## Запуск

```
flutter pub get
flutter run
```
