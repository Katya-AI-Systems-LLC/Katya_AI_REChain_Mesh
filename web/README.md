# Web Platform Development

This directory contains the web-specific implementation for the Katya AI REChain Mesh project using Flutter.

## Structure

- **index.html**: Main HTML file for the web app.
- **manifest.json**: Web app manifest for PWA features.
- **favicon.png**: Favicon.
- **icons/**: Icons for the web app.

## Development Setup

1. **Prerequisites**:
   - Install Flutter SDK.
   - Ensure web support is enabled (`flutter config --enable-web`).

2. **Building the App**:
   - Run `flutter pub get`.
   - Use `flutter build web` to build.
   - Serve with `flutter run -d chrome` or deploy to a web server.

3. **Features**:
   - Blockchain integration via web-compatible plugins.
   - Gaming and IoT modules in web context.
   - Social features using web APIs.

## Customization

- Edit `index.html` for custom HTML.
- Update `manifest.json` for PWA settings.
- Modify `lib/` for app logic.

## Documentation

See Flutter web documentation and project README.

## Testing

- Use `flutter test`.
- Test in browsers for compatibility.
