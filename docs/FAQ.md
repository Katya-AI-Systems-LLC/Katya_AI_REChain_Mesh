# FAQ (RU/EN)

**Q: Можно использовать без интернета?**
A: Да, mesh полностью оффлайн через Bluetooth/Wi-Fi/Nearby/UDP.

**Q: Как настроить mirror на SourceCraft/GitFlic?**
A: Добавьте remote (см. push_mirrors.sh) или импортируйте через веб-интерфейс.

**Q: Приложение не видит устройства/ошибка BLE?**
A: Проверьте разрешения, перезапустите, выберите mesh adapter в Settings.

**Q: Как протестировать AI и голосование?**
A: Зайдите на страницу Voting (главное меню), попробуйте создать poll и поговорите с AI-компаньоном.

**Q: Возникла ошибка запуска/сборки?**
A: Используйте `flutter clean`, обновите Flutter, смотрите TROUBLESHOOTING.md.
