# GitFlic Configuration

This folder contains configurations for GitFlic, a domestic Russian Git hosting platform.

## Contents

- **gitflic-ci.yml**: CI/CD pipeline for GitFlic.
- **gitflic-ci-clean.yml**: Clean version of the pipeline.
- **gitflic.yml**: General configuration.
- **Other files**: Additional configs.

## Setup

1. Copy the relevant files to your GitFlic repository.
2. Customize the CI/CD pipeline.
3. Enable CI/CD in your GitFlic project settings.

Refer to GitFlic documentation for more info.
