# SourceCraft Configuration

This folder contains configurations for SourceCraft, a domestic Russian Git hosting platform.

## Contents

- **sourcecraft-ci.yml**: CI/CD pipeline for SourceCraft.
- **sourcecraft-ci-clean.yml**: Clean version of the pipeline.
- **sourcecraft.yml**: General configuration.
- **Other files**: Additional configs.

## Setup

1. Copy the relevant files to your SourceCraft repository.
2. Customize the CI/CD pipeline based on your project.
3. Ensure CI/CD is enabled in your SourceCraft project settings.

For more details, visit SourceCraft documentation.
