# GitVerse Configuration

This folder contains configurations for GitVerse, a domestic Russian Git hosting platform.

## Contents

- **gitverse-ci.yml**: CI/CD pipeline for GitVerse.
- **gitverse-ci-clean.yml**: Clean version of the pipeline.
- **gitverse.yml**: General configuration.
- **Other files**: Additional configs.

## Setup

1. Copy the relevant files to your GitVerse repository.
2. Customize the CI/CD pipeline.
3. Enable CI/CD in your GitVerse project settings.

Refer to GitVerse documentation for more info.
