# Bitbucket Configuration

This folder contains Bitbucket-specific configurations, including Pipelines and other features.

## Contents

- **bitbucket-pipelines.yml**: CI/CD pipeline for Bitbucket.
- **Other files**: Additional configs.

## Setup

1. Copy `bitbucket-pipelines.yml` to the root of your Bitbucket repository.
2. Customize the pipeline as needed.
3. Enable Pipelines in your Bitbucket repository settings.

Refer to Bitbucket Pipelines documentation for more info.
