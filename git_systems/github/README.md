# GitHub Configuration

This folder contains GitHub-specific configurations, including Actions workflows, issue templates, and other GitHub features.

## Contents

- **workflows/**: GitHub Actions workflows for CI/CD.
- **ISSUE_TEMPLATE/**: Templates for issues.
- **PULL_REQUEST_TEMPLATE.md**: Template for pull requests.
- **Other files**: Configs like codeql, dependabot.

## Setup

1. Copy the contents of this folder to `.github/` in your repository.
2. Customize workflows as needed for your project.
3. Ensure GitHub Actions are enabled in your repository settings.

For more details, refer to GitHub documentation.
