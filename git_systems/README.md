# Git Systems Configuration

This directory contains configurations and setups for various Git hosting platforms, including international and domestic systems.

## Supported Systems

- **GitHub**: Standard GitHub Actions and workflows.
- **GitLab**: CI/CD pipelines for GitLab.
- **Bitbucket**: Pipelines for Bitbucket.
- **Gitea**: Workflows for Gitea.
- **Domestic Russian Systems**:
  - SourceCraft
  - GitFlic
  - GitVerse
- **International Systems** (by country):
  - Canada
  - Israel
  - Arab countries
  - Australia
  - China
  - And more...

## Structure

Each subfolder or file corresponds to a specific Git system with its CI/CD configurations, documentation, and setup guides.

## Usage

Refer to individual README files in each system's folder for detailed setup instructions.
